# 🚀 GUIDE RAPIDE - Portfolio en 5 minutes

## Étape 1: Ouvrir le portfolio
```
Double-cliquez sur index.html pour le voir dans votre navigateur
```

## Étape 2: Personnaliser le texte principal (5 min)

Ouvrez `index.html` et trouvez ces lignes (autour de la ligne 42):

```html
<h1 class="hero-title">Hicham Ziate</h1>
<p class="hero-subtitle">Data Analyst & IA Specialist</p>
<p class="hero-description">Transforming data into insights | Machine Learning & Analytics Enthusiast</p>
```

**Remplacez par vos infos:**
```html
<h1 class="hero-title">VOTRE NOM</h1>
<p class="hero-subtitle">VOTRE TITRE (ex: Data Scientist)</p>
<p class="hero-description">VOTRE COURTE BIO</p>
```

Sauvegardez avec Ctrl+S et rechargez la page dans le navigateur.

## Étape 6: Télécharger le CV (1 min)

Cherchez cette ligne:
```html

```

Remplacez le chemin par le chemin de VOTRE CV (garder `/` même sur Windows en HTML).

## Étape 7: Déployer en ligne (5 min) ⭐

### Option A: Netlify (PLUS FACILE)
1. Allez sur [netlify.com](https://netlify.com)
2. Créez un compte gratuit
3. Glissez-déposez le dossier `FormationMed5Sale`
4. ✅ Votre site est LIVE!

### Option B: GitHub → GitHub Pages
1. Créez un repository sur GitHub
2. Uploadez les 3 fichiers (index.html, styles.css, app.js)
3. Settings → Pages → Choisir main
4. Accédez à `username.github.io/FormationMed5Sale`
